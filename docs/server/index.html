Python package server

PyPi-like package server for the MetPy packages

WARNING: This package is under active development. The current version is considered non-stable.

This website server as a PyPi-like package server for the MetPy packages. It conforms to Pep 503. It is based on a blog post by freeCodeCamp.

The packages available in this server can be used to specify dependencies in pyproject.toml files including the entry dependencies = ["package-name"] They can also be installed via pip:
pip install package_name --extra-index-url https://xandratxan.github.io/python-package-server/

The following packages are available:

Package Description
physical-magnitude Simple operations with magnitudes including units and uncertainties
